# h-player-v2

## 关于h-player-v2

h-player-v2是基于Quasar Framework开发应用程序，通过electron包装，实现对`色色资源站`，`玖玖资源站`，`撸死你资源站`，`富二代资源站`，`字幕网资源站`，`久草资源站`，`女优资源站`，`利来资源站`的分类浏览与在线播放。

## Get Started
```bash
# 安装依赖
yarn install
# 启动开发服务器
yarn electron:serve
# 编译二进制包
yarn electron:build
```

## windows release
